# CARINA Sandbox - Python Template

## Purpose

Python sandbox for data science, scripts, and AI experimentation. This template provides Python 3.12 with pip and venv support.

## What it's for

- Running Python scripts
- Data science experiments
- Testing AI/ML code
- Package testing with pip

## What it's NOT for

- Production deployments
- Long-running services
- Database operations
- Web frameworks (install separately if needed)

## Included

- Python 3.12
- pip (latest)
- venv support

## Security

- Runs as unprivileged user `sandbox`
- No host filesystem access
- No privileged capabilities
- Packages install to user directory only
