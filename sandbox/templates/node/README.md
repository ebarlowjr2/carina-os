# CARINA Sandbox - Node.js Template

## Purpose

Node.js sandbox for tooling and UI experiments. This template provides Node.js LTS with npm support.

## What it's for

- Running Node.js scripts
- Testing npm packages
- UI/tooling experiments
- Build tool testing

## What it's NOT for

- Production deployments
- Long-running servers
- Database operations
- Global package installations

## Included

- Node.js LTS
- npm

## Security

- Runs as unprivileged user `sandbox`
- No host filesystem access
- No privileged capabilities
- Global packages install to user directory only
