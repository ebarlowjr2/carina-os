# CARINA Wallpapers

Place wallpaper files in this directory.

## Required Files

- `carina-void.png` - Primary CARINA wallpaper (deep space theme)

## Usage

The bootstrap script and GUI enable command will set the wallpaper automatically when available.

## Specifications

- Resolution: 3840x2160 (4K) recommended, will scale to other resolutions
- Format: PNG preferred
- Theme: Deep space, matches CARINA color palette (see ../colors.md)
- No logos in Sprint 1
