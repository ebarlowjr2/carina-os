What is CARINA OS

CARINA OS is a mission-focused operating system built on Ubuntu, designed for engineers, scientists, and developers who need reliable, secure, and efficient computing environments.

Core Components

Component Description
CARINA Core Headless-first base system with CLI tools
FlightDeck Optional GUI environment (GNOME-based)
Sandbox Isolated execution environments using Podman
MissionLab Embedded and robotics development tooling

Quick Start

Installation

git clone https://github.com/ebarlowjr2/carina-os.git
cd carina-os
sudo ./bootstrap/bootstrap-carina.sh

Verify Installation

carina doctor

Enable GUI (FlightDeck)

sudo carina gui enable

Disable GUI

sudo carina gui disable

Update CARINA

cd ~/carina-os
git pull
sudo ./bootstrap/bootstrap-carina.sh

Sandbox Quick Start

CARINA Sandbox provides isolated execution environments for testing code safely.

List Available Templates

carina sandbox templates

Start a Sandbox

carina sandbox up python --ttl 30m

Execute Commands

carina sandbox exec <sandbox-id> python --version

List Active Sandboxes

carina sandbox list

Stop a Sandbox

carina sandbox down <sandbox-id>

Cleanup Expired Sandboxes

carina sandbox cleanup

MissionLab Overview

MissionLab provides tools for embedded development and robotics.

Check Status

carina missionlab status

Detect Connected Devices

carina missionlab devices

Install Tools

sudo carina missionlab install arduino-cli
sudo carina missionlab install platformio

List Available Tools

carina missionlab list

Apply Embedded Profile

sudo carina profile apply missionlab-embedded
Note: After installing PlatformIO, add ~/.local/bin to your PATH: 
export PATH="$PATH:$HOME/.local/bin"

Security Model

Sandbox Security

  • All containers run unprivileged
  • All capabilities dropped (--cap-drop ALL)
  • Read-only filesystem with tmpfs for /tmp
  • Resource limits enforced (512MB RAM, 1 CPU)
  • No host mounts by default

User Groups

Group Purpose
carina Access to CARINA state and logs
dialout Serial port access
plugdev USB device access
